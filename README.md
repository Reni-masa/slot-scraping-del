# 【slot-scraping】

# 基本機能
スロットデータの収集をwebスクレイピングを利用して行うツールです。

herokuを使用し、定時刻(23:30)にデータ収集を自動化。

スクレイピングしたデータに対して評価をし、一式のデータをDBにため込むというシンプルな機能。

# 使用技術
言語：python

ライブラリ：

DB：mysql

インフラ：heroku
